# OMC Invoice Manager - Simple PWA

Simple, offline-capable invoice manager for Owensboro Mowing Company.

## Features

- **Super Simple**: Just 3 tabs - Clients, New Invoice, Settings
- **Offline-First**: Works completely offline after installation
- **Client Management**: Store client names, phone numbers, emails, and addresses
- **Quick Invoicing**: Add line items with description, quantity, and price
- **PDF Generation**: Create professional PDFs instantly
- **Share via SMS**: Send invoices via text message on iOS/Android
- **PWA Installable**: Install on your phone like a real app
- **Data Sync**: Export/import to move data between devices

## How to Use

### Simple Workflow

1. **Clients Tab**: Add your clients (name, phone, email, address)
2. **New Invoice Tab**:
   - Select a client
   - Add line items (description, qty, price)
   - Click "Share via Text" (orange button)
3. **Settings Tab**: Set up your company info

That's it!

## Install on Your Phone

**iPhone:**
1. Open in Safari
2. Tap Share button
3. Tap "Add to Home Screen"

**Android:**
1. Open in Chrome
2. Tap "Install App" when prompted

## Sync Between Phone & Laptop

**Export from one device:**
- Settings tab → Export Data
- Email the JSON file to yourself

**Import on other device:**
- Settings tab → Import Data
- Select the JSON file
