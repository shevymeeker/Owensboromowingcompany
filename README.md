# Owensboro Mowing Company - Modern CRM

A clean, modern, and user-friendly business management system for Owensboro Mowing Company.

## Features

### Dashboard
- Real-time business overview
- Revenue tracking
- Customer count
- Today's jobs at a glance
- Recent jobs list

### Customer Management
- Add, edit, and delete customers
- Store contact information (phone, email, address)
- Add notes for each customer
- Fast search functionality

### Schedule
- Interactive calendar view
- Visual job tracking
- See jobs by date
- Monthly navigation
- Today's schedule at a glance

### Job Management
- Create and track all jobs
- Multiple service types (mowing, edging, trimming, etc.)
- Job status tracking (scheduled, in-progress, completed, cancelled)
- Price management
- Filter by status
- Quick status updates

## Technology

Built with modern 2025 web technologies:
- **React 18** - Fast, modern UI framework
- **Vite** - Lightning-fast development and builds
- **Tailwind CSS** - Beautiful, responsive design
- **Lucide Icons** - Clean, modern icons
- **Local Storage** - Simple data persistence (no complex backend needed)

## Getting Started

### Install Dependencies
```bash
npm install
```

### Run Development Server
```bash
npm run dev
```

### Build for Production
```bash
npm run build
```
